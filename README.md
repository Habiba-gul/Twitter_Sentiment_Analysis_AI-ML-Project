# TwitSent - Twitter Sentiment Analysis



## Project Overview

  Twitter Sentiment Analysis is a full-stack Machine Learning project that analyzes the sentiment of any tweet instantly. 

Users can type or paste a tweet, and the application predicts whether the sentiment is **Positive** or **Negative**, along with a confidence score and 5-star rating.

This project demonstrates end-to-end skills from data preprocessing to deployment.

## Features

- Real-time sentiment prediction using trained ML model
- Modern  UI with Twitter-inspired design
- Confidence percentage display
- 5-Star sentiment strength rating
- Hybrid prediction system (ML Model + Keyword Boosting) for better positive detection


## 📊 Model Performance

- **Overall Accuracy-93.18%**
- Excellent performance on Negative tweets 
- Positive tweet detection improved using hybrid rule-based boosting
